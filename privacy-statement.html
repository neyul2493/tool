PRIVACY STATEMENT

Your privacy is important to FREEFORMATTER. This privacy statement provides information about the Personal Information that FREEFORMATTER collects, and the ways in which FREEFORMATTER uses that Personal Information.

Personal Information collection

FREEFORMATTER may collect and use Personal Information about your use of this website, namely information about browsing and linking ("Personal Information");

The Tools available on FREEFORMATTER, namely, formatters, validators, encoders & decoders, code minifiers and beautifiers, converters, cryptography and security tools, string escapers and utilities, data generators and other web resources available on www.freeformatter.com (the "Tools") that require server side processing keep all the data in 'transient memory (RAM)' for a very short period of time. This data is not collected, stored or otherwise persisted.

Exceptionally, the "Batch Formatter" tool, which processes large files, requires the data to be temporarily stored on the cloud. This data is temporarily stored for a duration of up to 24 hours. There are policies in place for the removal of any data that persists on the cloud servers above this transient period. However, FREEFORMATTER does not make any representation or guarantee that these policies are applied as they are handled by the cloud server providers.
