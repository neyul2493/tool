What are the ISO-3166-1 codes for Turkey?

  • Alpha-2: TR
  • Alpha-3: TUR
  • Numeric: 792
  • Java Locale Code: tr_TR
  • .Net CultireInfo Code: tr-TR
  • PHP Locale Code: tr_TR

What is the official language in Turkey?

  • Turkish

What is the date format in Turkey?

The date format in Turkey is little-endian:

  • Format: dd.mm.yyyy
    Ex: 03.12.2014
How to format dates for Turkey in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("tr", "TR");
SimpleDateFormat sdf = new SimpleDateFormat("dd.MM.yyyy", locale);
sdf.format(new Date());
C#:
CultureInfo ci = CultureInfo.GetCultureInfo("tr-TR");
DateTime.Now.ToString("dd.MM.yyyy", ci);
JavaScript:
var date = new Date();
date.toLocaleDateString('tr-TR');
PHP:
date("d.m.Y");

What is the time format in Turkey?

The time format in Turkey is mostly 24-hour notation, but 24-hour format is also commonly adopted.

  • HH:mm[:ss]
    Ex: 21:00 for 09:00PM, and 09:00 for 09:00AM
How to format time for Turkey in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("tr", "TR");
SimpleDateFormat sdf = new SimpleDateFormat("HH:mm:ss", locale);
sdf.format(new Date());
C#:
CultureInfo ci = CultureInfo.GetCultureInfo("tr-TR");
DateTime.Now.ToString("HH:mm:ss", ci);
JavaScript:
var date = new Date();
date.toLocaleTimeString('tr-TR', {hour12: false});
PHP:
date("H:i:s");

What is the numeric format in Turkey?

    Format: 999.999.999,99

  • Group Size: 3
  • Grouping Character: . (Dot)
  • Decimal Character: , (Comma)
How to format numbers for Turkey in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("tr", "TR");
NumberFormat numberFormat = NumberFormat.getNumberInstance(locale);
numberFormat.format(999999999.99d);
C#:
double d = 999999999.99d;
d.ToString("n", CultureInfo.GetCultureInfo("tr-TR")));
JavaScript:
var number = 999999999.99;
number.toLocaleString('tr-TR');
PHP:
$fmt = new NumberFormatter($locale = 'tr_TR', NumberFormatter::DECIMAL);
$fmt->format(999999999.99);

What is the currency format in Turkey?

    Format: 999.999.999,99 ₺

  • Group Size: 3
  • Grouping Character: . (Dot)
  • Decimal Character: , (Comma)
  • Currency Symbol: ₺
  • Currency Symbol Position: After number
  • Currency Name: Turkish Lira (TRY)
How to format currencies for Turkey in Java, C#, PHP, and JavaScript 
Locale locale = new Locale("tr", "TR");
NumberFormat numberFormat = NumberFormat.getCurrencyInstance(locale);
numberFormat.format(999999999.99d);
C#:
double d = 999999999.99d;
d.ToString("c", CultureInfo.GetCultureInfo("tr-TR")));
JavaScript:
var number = 999999999.99;
number.toLocaleString('tr-TR', {currency: 'TRY', style: 'currency'});
PHP:
$fmt = new NumberFormatter($locale = 'tr_TR', NumberFormatter::CURRENCY);
$fmt->format(999999999.99);

  code VARCHAR(6) NOT NULL,
  name VARCHAR(50) NOT NULL,
  PRIMARY KEY (id),
  UNIQUE INDEX id_UNIQUE (id ASC),
  UNIQUE INDEX code_UNIQUE (code ASC)
) ENGINE = InnoDB;
