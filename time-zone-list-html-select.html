Time Zone List - HTML select/dropdown snippet

Included in this page is the HTML select/dropdown code snippets to generate a list of time zones. It's very hard to come up with a list of time zones that make sense because of considerations for countries and daylight savings adjustment. The list I use is based on a MySQL time zone list. If you want to use something else for the "value" attribute, feel free to replace it with something more meaningful to your needs, I often use the database identifier instead.

Depending on the projects where I use this list, I will make use of the different custom HTML attributes that are available on each option element. For example, a system that only cares for the GMT adjustment will make use of that attribute only.

Note that if you are NOT taking "Daylight Saving Time" into consideration when computing the offset from GMT, you might not end up with the proper time being displayed.

Custom option attributes:
  • timeZoneId: Corresponds to an internal time zone unique identifier, usually a database table entry.
  • gmtAdjustment: The GMT adjustment.
  • useDaylightTime: Indicates if the time zone makes use of "Daylight Saving Time". 1 is true, 0 is false.
  • value: The time GMT adjustment express as a integer value. So from GMT, -5 means, minus 5 hours.
