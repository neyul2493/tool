What is the date format in United Kingdom?

The date format in United Kingdom is little-endian:

  • Format: dd/MM/yyyy
    Ex: 31/12/2014 for December 31st 2014. Leading zeroes can be omitted, but not on the year field, hence d/M/yy is also acceptable
How to format dates for United Kingdom in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("en", "GB");
SimpleDateFormat sdf = new SimpleDateFormat("dd/MM/yyyy", locale); // or d/M/yy
sdf.format(new Date());
C#:
CultureInfo ci = CultureInfo.GetCultureInfo("en-GB");
DateTime.Now.ToString("dd/MM/yyyy", ci); // or d/M/yy
JavaScript:
var date = new Date();
date.toLocaleDateString('en-GB');
PHP:
date("d/m/Y");

What is the time format in United Kingdom?

The time format in United Kingdom is mostly 24-hour notation, but 24-hour format is also commonly adopted.

  • HH:mm[:ss]
    Ex: 21:00 for 09:00PM, and 09:00 for 09:00AM
How to format time for United Kingdom in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("en", "GB");
SimpleDateFormat sdf = new SimpleDateFormat("HH:mm:ss", locale);
sdf.format(new Date());
C#:
CultureInfo ci = CultureInfo.GetCultureInfo("en-GB");
DateTime.Now.ToString("HH:mm:ss", ci);
JavaScript:
var date = new Date();
date.toLocaleTimeString('en-GB', {hour12: false});
PHP:
date("H:i:s");

What is the numeric format in United Kingdom?

    Format: 999,999,999.99

  • Group Size: 3
  • Grouping Character: , (Comma)
  • Decimal Character: . (Dot)
How to format numbers for United Kingdom in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("en", "GB");
NumberFormat numberFormat = NumberFormat.getNumberInstance(locale);
numberFormat.format(999999999.99d);
C#:

JavaScript:
var number = 999999999.99;
number.toLocaleString('en-GB');
PHP:
$fmt = new NumberFormatter($locale = 'en_GB', NumberFormatter::DECIMAL);
$fmt->format(999999999.99);

What is the currency format in United Kingdom?

    Format: £999,999,999.99

  • Group Size: 3
  • Grouping Character: , (Comma)
  • Decimal Character: . (Dot)
  • Currency Symbol: £
  • Currency Symbol Position: Before number
  • Currency Name: United Kingdom Pound (GBP)
How to format currencies for United Kingdom in Java, C#, PHP, and JavaScript 
Locale locale = new Locale("en", "GB");
NumberFormat numberFormat = NumberFormat.getCurrencyInstance(locale);

d.ToString("c", CultureInfo.GetCultureInfo("en-GB")));

number.toLocaleString('en-GB', {currency: 'GBP', style: 'currency'});
PHP:
$fmt = new NumberFormatter($locale = 'en_GB', NumberFormatter::CURRENCY);
$fmt->format(999999999.99);
