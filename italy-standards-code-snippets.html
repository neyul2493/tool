  • Alpha-3: ITA

  • Java Locale Code: it_IT

  • PHP Locale Code: it_IT

What is the official language in Italy?

  • Italian

What is the date format in Italy?

The official date format is little-endian, but little-endian is often used:

  • Format: dd/mm/yyyy
    Ex: 03/12/2014 for December 31st 2014
How to format dates for Italy in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("it", "IT");
SimpleDateFormat sdf = new SimpleDateFormat("dd/MM/yyyy", locale);
sdf.format(new Date());
C#:
CultureInfo ci = CultureInfo.GetCultureInfo("it-IT");
DateTime.Now.ToString("dd/MM/yyyy", ci);
JavaScript:
var date = new Date();
date.toLocaleDateString('it-IT');
PHP:
date("d/m/Y");

What is the time format in Italy?

The time format in Italy is mostly 24-hour notation, but 24-hour format is also commonly adopted.

  • HH:mm[:ss]
    Ex: 21:00 for 09:00PM, and 09:00 for 09:00AM
How to format time for Italy in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("it", "IT");
SimpleDateFormat sdf = new SimpleDateFormat("HH:mm:ss", locale);
sdf.format(new Date());
C#:
CultureInfo ci = CultureInfo.GetCultureInfo("it-IT");
DateTime.Now.ToString("HH:mm:ss", ci);
JavaScript:
var date = new Date();
date.toLocaleTimeString('it-IT', {hour12: false});
PHP:
date("H:i:s");

What is the numeric format in Italy?

    Format: 999.999.999,99

  • Group Size: 3
  • Grouping Character: . (Dot)
  • Decimal Character: , (Comma)
How to format numbers for Italy in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("it", "IT");
NumberFormat numberFormat = NumberFormat.getNumberInstance(locale);
numberFormat.format(999999999.99d);
C#:
double d = 999999999.99d;
d.ToString("n", CultureInfo.GetCultureInfo("it-IT")));
JavaScript:
var number = 999999999.99;
number.toLocaleString('it-IT');
PHP:
$fmt = new NumberFormatter($locale = 'it_IT', NumberFormatter::DECIMAL);
$fmt->format(999999999.99);

What is the currency format in Italy?

    Format: 999.999.999,99 €

  • Group Size: 3
  • Grouping Character: . (Dot)
  • Decimal Character: , (Comma)
  • Currency Symbol: €
  • Currency Symbol Position: After number
  • Currency Name: Euro (EUR)
How to format currencies for Italy in Java, C#, PHP, and JavaScript 
Locale locale = new Locale("it", "IT");
NumberFormat numberFormat = NumberFormat.getCurrencyInstance(locale);
numberFormat.format(999999999.99d);
C#:
double d = 999999999.99d;
d.ToString("c", CultureInfo.GetCultureInfo("it-IT")));
JavaScript:
var number = 999999999.99;
number.toLocaleString('it-IT', {currency: 'EUR', style: 'currency'});
PHP:
$fmt = new NumberFormatter($locale = 'it_IT', NumberFormatter::CURRENCY);
$fmt->format(999999999.99);
