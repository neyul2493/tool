  • PHP Locale Code: vi_VN

What is the official language in Vietnam?

What is the date format in Vietnam?

The date format in Vietnam is little-endian:

  • Format: dd/MM/yyyy
    Ex: 31/12/2014 for December 31st 2014. Leading zeroes can be omitted, but not on the year field, hence d/M/yy is also acceptable
How to format dates for Vietnam in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("vi", "VN");
SimpleDateFormat sdf = new SimpleDateFormat("dd/MM/yyyy", locale);
sdf.format(new Date());
C#:
CultureInfo ci = CultureInfo.GetCultureInfo("vi-VN");
DateTime.Now.ToString("dd/MM/yyyy", ci);
JavaScript:
var date = new Date();
date.toLocaleDateString('vi-VN');
PHP:
date("d/m/Y");

What is the time format in Vietnam?

The time format in Vietnam is 12-hour notation. The leading zero in hours can be omitted:

  • h:mm[:ss] SA/CH
    Ex: 9:00 CH for 09:00PM, and 9:00 SA for 09:00AM
How to format time for Vietnam in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("vi", "VN");
SimpleDateFormat sdf = new SimpleDateFormat("h:mm:ss a", locale);
sdf.format(new Date());
C#:
CultureInfo ci = CultureInfo.GetCultureInfo("vi-VN");
DateTime.Now.ToString("h:mm:ss tt", ci);
JavaScript:
var date = new Date();
date.toLocaleTimeString('vi-VN', {hour12: true});
PHP:
date("g:i:s a");

What is the numeric format in Vietnam?

    Format: 999.999.999,99

  • Group Size: 3
  • Grouping Character: . (Dot)
  • Decimal Character: , (Comma)
How to format numbers for Vietnam in Java, C#, PHP, and JavaScript
Java:
Locale locale = new Locale("vi", "VN");
NumberFormat numberFormat = NumberFormat.getNumberInstance(locale);
numberFormat.format(999999999.99d);
C#:
double d = 999999999.99d;
d.ToString("n", CultureInfo.GetCultureInfo("vi-VN")));
JavaScript:
var number = 999999999.99;
number.toLocaleString('vi-VN');
PHP:
$fmt = new NumberFormatter($locale = 'vi_VN', NumberFormatter::DECIMAL);
$fmt->format(999999999.99);

What is the currency format in Vietnam?

    Format: 1.000.000.000 đ

  • Group Size: 3
  • Grouping Character: . (Dot)
  • Decimal Character: Not supported
  • Currency Symbol: ₫
  • Currency Symbol Position: Ater number
  • Currency Name: Vietnamese dong (VND)
How to format currencies for Vietnam in Java, C#, PHP, and JavaScript 
Locale locale = new Locale("vi", "VN");
NumberFormat numberFormat = NumberFormat.getCurrencyInstance(locale);
numberFormat.format(1000000000d);
C#:
double d = 1000000000d;
d.ToString("c", CultureInfo.GetCultureInfo("vi-VN")));
JavaScript:
var number = 1000000000;
number.toLocaleString('vi-VN', {currency: 'VND', style: 'currency'});
PHP:
$fmt = new NumberFormatter($locale = 'vi_VN', NumberFormatter::CURRENCY);
$fmt->format(1000000000);
